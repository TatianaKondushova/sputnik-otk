# Релиз Сервис ОТК v0.3.0

**Дата:** 2026-06-05  
**Версия:** 0.3.0 (build 3)  
**Приложение:** `ru.sputnik.otk`  
**Устанавливается параллельно со старым APK (`com.example.my_application`)**

---

## ✅ Что реализовано

### NFC-сканирование
- Автоматическое чтение NFC-меток панелей (NDEF text/plain)
- Ручной ввод 10-значного номера панели
- Защита от дубликатов (вибрация + Toast)
- Автонавигация на экран ОТК при сканировании с главного экрана

### Список проверки
- Отображение всех отсканированных панелей
- Поле «Неисправность» к каждой панели (редактирование через иконку ✏)
- Автозаполнение «нет», если комментарий пустой
- Очистка всего списка (иконка 🧹)
- Копирование списка в буфер обмена (иконка 📋)

### Отправка данных
- POST-запрос в Google Apps Script webhook
- Передаётся: пароль, имя листа, мастер, массив панелей с номерами и неисправностями
- Индикатор прогресса отправки
- Обработка ошибок сети с виброоткликом

### Выбор мастера
- Выпадающий список мастеров
- Сохраняется на сессию (не между перезапусками приложения)

### Настройки
- Смена webhook URL
- Смена пароля доступа
- Редактирование списка мастеров
- Доступ: долгое нажатие на заголовок экрана ОТК

### Логи
- Просмотр журнала ошибок (`error_log.json`)
- Кнопка очистки логов

### UI/UX
- Фирменная синяя тема + Material 3
- Крупные шрифты для удобства на производстве
- Зелёная кнопка «Сохранить»
- Вибрация: короткая при успехе, длинная при ошибке/дубликате

## 🚀 Сборка

```bash
# Debug APK
./gradlew assembleDebug

# Release APK (keystore настроен)
./gradlew assembleRelease
```

**Keystore:** `app/sputnik-otk.keystore`  
**Alias:** `sputnik-otk`  
**ProGuard:** включён (`isMinifyEnabled = true`, `isShrinkResources = true`)

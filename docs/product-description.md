# Описание приложения «Сервис ОТК»

**Дата:** 17.04.2026
**Автор:** Кондушова Татьяна
**Статус:** на согласовании

---

## 1. Назначение

Мобильное Android-приложение для контроллёров ОТК. Позволяет сканировать панели через NFC и записывать результаты проверок в Google-таблицу.

## 2. Что делает приложение

Контроллёр ОТК проверяет панели на стенде, затем сканирует NFC-метки проверенных панелей телефоном. Приложение собирает список отсканированных панелей и отправляет данные в Google-таблицу одним нажатием кнопки «Сохранить».

### Данные, которые записываются в таблицу

| Колонка | Что записывается | Как заполняется |
|---------|-----------------|-----------------|
| A — Номер панели | Номер с NFC-метки или ручной ввод | Автоматически при сканировании |
| B — Мастер | Имя контроллёра | Выбирается из списка |
| C — Дата | Текущая дата | Автоматически |
| D — Неисправность | Комментарий о дефекте | Вручную, если нужно |
| E — Замечание | — | Заполняется Татьяной позже |
| F — Что тестировали | — | Заполняется Татьяной позже |

### Рабочий сценарий

1. Контроллёр открывает приложение, выбирает своё имя из списка. Без выбора имени нельзя добавлять панели. Выбор действует до закрытия приложения; после перезапуска нужно выбрать заново.
2. Устанавливает партию панелей на стенд (до 8 штук), выполняет проверку.
3. Подносит телефон к NFC-метке каждой панели — номер появляется в списке на экране. Если метка не читается — вводит номер вручную.
4. Если есть дефект — нажимает значок редактирования, вписывает комментарий.
5. Нажимает «Сохранить» — все панели из списка отправляются в таблицу.
6. Берёт следующую партию, повторяет.

За смену один контроллёр проверяет **до 200 панелей** (около 25 партий).

## 3. Пользователи

- **4 штатных контроллёра ОТК:** Руслан, Камиль, Виктор, Тимур.
- **«Мастер»** — пятый пункт выпадающего списка. Используется, если проверкой занялся незапланированный человек (например, руководитель, который вне графика подменил контроллёра).
- **Татьяна (администратор)** — имеет доступ к скрытому экрану настроек для смены адреса сервера и пароля без переустановки приложения.

## 4. Экраны приложения

### 4.1. Главный экран

Заголовок «Сервис ОТК» и одна кнопка-плитка «ОТК». В будущем сюда можно добавить плитку «SK-Warranty» без переделки приложения.

### 4.2. Экран ОТК (основной рабочий экран)

- Выбор мастера (выпадающий список). Обязателен — пока имя не выбрано, добавление панелей (и по NFC, и вручную) недоступно. Выбор не сохраняется между запусками: после перезапуска приложения его нужно сделать заново.
- Текущая дата (определяется автоматически).
- Поле ввода номера панели + кнопка добавления.
- Список отсканированных панелей с возможностью добавить комментарий к каждой.
- Кнопки: «Сохранить» (отправка в таблицу), «Логи» (журнал ошибок), «Назад».
- Дополнительно: очистка списка, копирование всех номеров в буфер обмена.
- Защита от дубликатов: если панель уже в списке, повторно не добавляется.

### 4.3. Экран логов

Журнал ошибок отправки (нет связи, неверный пароль и т.д.). Помогает понять, какие панели не отправились и почему.

### 4.4. Экран настроек (скрытый)

Доступен только через долгое нажатие (3 секунды) на заголовок. Позволяет изменить адрес сервера и пароль без пересборки приложения.

## 5. Технические требования

- **Платформа:** Android 8.0 и выше.
- **Телефоны сотрудников:** Huawei P30 Lite и аналогичные с поддержкой NFC.
- **Интернет:** Wi-Fi всегда доступен на рабочих местах. Офлайн-режим не требуется.
- **Способ отправки данных:** Google Apps Script (веб-сервис внутри Google-таблицы). Не требует отдельного сервера или администратора.

## 6. Что НЕ входит в текущую версию

- Блок SK-Warranty (работа с Bitrix) — планируется отдельно, вторым этапом.
- Авторизация (логин/пароль при входе в приложение).
- Аналитика и отчёты.
- Версия для iPhone.

## 7. Этапы разработки

Каждый этап заканчивается работающей версией приложения, которую можно установить на телефон и проверить.

| Этап | Что появляется | Статус |
|------|---------------|--------|
| 1. Каркас | Приложение открывается, показывает экран-заглушку | Готово |
| 2. Навигация | Главный экран с плиткой «ОТК», переход на экран ОТК | В работе |
| 3. Базовый ОТК | Выбор мастера, ввод номера, список, отправка в таблицу | — |
| 4. NFC | Сканирование меток телефоном | — |
| 5. Редактирование | Комментарии к панелям, очистка списка, копирование | — |
| 6. Логи | Журнал ошибок отправки | — |
| 7. Настройки | Скрытый экран смены адреса и пароля | — |
| 8. Финал | Иконка, финальная проверка, раздача сотрудникам | — |

## 8. Распространение

APK-файл передаётся на телефоны сотрудников через USB, мессенджер или облако. Установка: разрешить «Установку из неизвестных источников» (один раз), открыть файл, нажать «Установить».

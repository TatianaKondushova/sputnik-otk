# Sputnik OTK

Android-приложение для записи результатов проверок ОТК в Google Sheets.

## Стек

- Kotlin + Jetpack Compose + Material 3
- minSdk 24 (Android 7.0), targetSdk 35
- Gradle 8.10.2, Kotlin 2.0.21, Compose BOM 2024.09.03

## Сборка

```bash
./gradlew assembleDebug
```

APK появится в `app/build/outputs/apk/debug/app-debug.apk`.

## Установка на телефон

Скачать APK из раздела [Releases](../../releases) и установить (потребуется разрешить установку из неизвестных источников).
